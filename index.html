World Wide Web [ W3C ]

World Wide Web (po angielsku: „ogólnoświatowa sieć”, „światowa rozległa sieć internetowa”), w skrócie Web lub częściej WWW – hipertekstowy, multimedialny, internetowy system informacyjny oparty na publicznie dostępnych, otwartych standardach IETF i W3C. WWW jest usługą internetową, która ze względu na zdobytą popularność bywa błędnie utożsamiana z całym Internetem
